# Rasberrypi_project

본 프로젝트는 라즈베리파이 보드를 이용하여 집안의 온도 습도 를 실시간으로 센싱해 간단한 홈페이지에서 결과를 보여주며 집안의 에어컨을 끄고 키며 희망온도도 바꿀 수 있게
하는 집안 온습도 제어 IOT입니다.
